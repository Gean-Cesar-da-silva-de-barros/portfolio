Sobre mim

Olá! Sou Gean, formado em Análise e Desenvolvimento de Sistemas, com foco em desenvolvimento web. Tenho experiência prática com HTML, CSS, JavaScript e Git/GitHub, aplicando conceitos de design responsivo, boas práticas de código e funcionalidades interativas em meus projetos.

Este portfólio foi criado para ser um reflexo do meu aprendizado e evolução na programação. Cada projeto aqui apresentado foi desenvolvido para demonstrar minhas habilidades técnicas, meu cuidado com a experiência do usuário e meu interesse em construir aplicações bem estruturadas, performáticas e de fácil manutenção.

Sou autodidata, persistente e organizado, sempre em busca de novas formas de melhorar meus conhecimentos e práticas. Meu objetivo é iniciar minha carreira como Front-End, Analista de Suporte de TI, Desenvolvedor de Sistemas ou em áreas correlatas, contribuindo para a criação de interfaces modernas e impactantes.

Assista a uma demonstração complementar do meu trabalho:

Habilidades

Exploro JavaScript para adicionar interatividade e lógica às páginas, trabalhando com eventos e manipulação do DOM para criar comportamentos dinâmicos e melhorar a experiência do usuário

Estruturo páginas com HTML semântico para manter o conteúdo bem organizado, acessível e fácil de manter, garantindo uma base sólida para a interface

Estilizo interfaces com CSS pensando em responsividade, hierarquia visual e consistência, ajustando layouts e detalhes para deixar o design limpo e agradável em diferentes telas

Com Tailwind, construo interfaces responsivas com agilidade usando classes utilitárias para controlar layout, espaçamentos, tipografia e cores diretamente no HTML, mantendo consistência visual e facilitando ajustes rápidos no design sem depender de muito CSS personalizado

Tenho utilizado o GitHub para versionar meus projetos com commits frequentes e mensagens claras, mantendo o repositório organizado e facilitando atualizações, testes e evolução do código ao longo do tempo

Uso Git para versionar projetos com mais segurança, mantendo um histórico claro de mudanças com commits objetivos e organização por branches quando necessário

No Figma, estruturo a ideia antes do código, definindo layout, tipografia e cores, além de montar protótipos simples para visualizar o fluxo e manter consistência no design

Com Bootstrap, acelero a construção de interfaces responsivas usando grid e componentes prontos, ajustando classes e detalhes visuais para deixar o resultado mais profissional e bem alinhado ao projeto
